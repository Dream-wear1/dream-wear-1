Why Add Analytics?

  • Track visitor count and behavior
  • See which products are popular
  • Monitor conversion rates
  • Understand your customers better

Option 1: Google Analytics (Free & Detailed)

Step 1: Create Google Analytics Account

  1. Go to analytics.google.com
  2. Sign in with your Google account
  3. Click "Start measuring"
  4. Create a new property for your website
  5. Enter your website URL (your GitHub Pages URL)

Step 2: Get Your Tracking Code

Google will provide you with a tracking code that looks like this:

<!-- Google Analytics --> <script async src="https://www.googletagmanager.com/gtag/js?id=GA_TRACKING_ID"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){dataLayer.push(arguments);} gtag('js', new Date()); gtag('config', 'GA_TRACKING_ID'); </script>

Step 3: Add to Your Website

Add the tracking code to the <head> section of your index.html file, right after the EmailJS script tag.

Option 2: Simple Counter (Easy)

StatCounter (Free with limitations)

  1. Visit statcounter.com
  2. Sign up for free account
  3. Add your website URL
  4. Get the HTML code
  5. Add to your website footer

What you'll get:

  • Visitor count
  • Page views
  • Countries/locations
  • Basic referral sources

Option 3: GitHub Pages Traffic (Built-in)

How to Check GitHub Traffic:

  1. Go to your repository on GitHub
  2. Click "Insights" tab
  3. Click "Traffic" in left sidebar
  4. View last 14 days of data

Limitations:

  • Only shows last 14 days
  • Basic metrics only
  • No real-time data
  • No detailed user behavior

For E-commerce Specifically

Enhanced E-commerce Tracking

Since you have an online store, you might want to track:

  • Product page views
  • Add to cart events
  • Checkout process
  • Order completions
  • Revenue tracking

Recommendation: Start with Google Analytics as it's free and provides comprehensive e-commerce tracking.
